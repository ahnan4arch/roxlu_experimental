Geometric Tools
===============
These are a couple of wrappers around GeometricTools which 
you can find at (http://www.geometrictools.com/). This is a massive
well coded library which contains a vast amount of geometric 
algorithm (voronoi, delaunay, convex hull, picking, clipping etc.).

How to use
===========
- Drag and drop the "Library" into XCode
- Add a search path to the Include dir.
- DO NOT DRAG&DROP THE INCLUDE INTO XCODE


Compile info
===========
Just download the zip and run "MacBuildWm5.sh" file.
